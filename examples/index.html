Examples

Pick the closest pattern, study it and apply it into your project.

Quick tips

All examples load ../AbrantesPlus.js (core + plugins). You should generate a custom build with the plugins you need using gulp.

Some examples include Google Tag Manager / GA4 snippets; replace IDs or remove tracking scripts as needed.
