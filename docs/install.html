Choose your build

Abrantes.js (core)

Use the lightweight core only. Add the plugins you need as separate scripts.

  • Smallest footprint.
  • You control which plugins are loaded.
  • Best for custom setups.

AbrantesPlus.js (core + plugins)

Prebuilt bundle with the current set of plugins from the repo.

  • Fastest way to get started.
  • Available as AbrantesPlus.js and AbrantesPlus.min.js.
  • Customize the plugin list with gulp.

Add the script to your page

Use AbrantesPlus

Recommended for quick installs and demos.

<script src="AbrantesPlus.min.js"></script>
<script>
  // Your experiment code here
</script>

Core + plugins

Load Abrantes first, then the plugins you need.

<script src="Abrantes.js"></script>
<script src="plugins/ga4-gtag.js"></script>
<script src="plugins/add2DL.js"></script>
<script>
  // Your experiment code here
</script>

Script order and dependencies

  • Load Abrantes before any plugin scripts.
  • Load analytics scripts (gtag, GTM, Hotjar, Clarity, Matomo) before calling their plugin methods.
  • Place your experiment code after the scripts or use defer on the script tags.
  • When testing locally, use a web server (not file://) so storage and redirects work correctly.
