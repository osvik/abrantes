Create experiments

Define variants, assign users, render changes, and track results.

Plugin reference View examples

Basic setup

Create a test object, define variants, then assign and render the variant.

var MyTest = Object.create(Abrantes);

MyTest.variants = [
  function () { /* control stays as the default page */ },
  function () { document.querySelector(".cta").textContent = "Start free trial"; }
];

MyTest.assignVariant("pricing_cta");
MyTest.renderVariant();
MyTest.persist("cookie");

If assignVariant returns -1, the user is not in the experiment and the page remains unchanged.

Traffic allocation and segmentation

Use the optional parameters in assignVariant to limit traffic or target a segment.

MyTest.assignVariant(
  "pricing_cta",
  0.5,
  function () { return window.innerWidth > 768; }
);
  • trafficAllocation is a number between 0 and 1.
  • segment is a function that returns true or false.

Tracking results

Tracking is handled by plugins. Call the plugin method after assignVariant.

// Example with the GA4 gtag plugin
MyTest.track("pricing_cta_event");
MyTest.trackUser("pricing_cta_user");

See the plugin reference for details and required analytics scripts.

Advanced patterns

Wait for the DOM

For SPAs or delayed content, wait for elements before rendering changes.

MyTest.waitFor(".pricing-table", function () {
  MyTest.renderVariant();
});

Cross-site experiments

Export the variant to another domain and import it on the destination.

// Origin
MyTest.crossSiteLink("a[data-cross-site]");

// Destination
MyTest.importVariant("pricing_cta");
MyTest.renderVariant();

Redirect tests

Use different pages for each variant and keep URL parameters.

if (MyTest.variant === 1) {
  MyTest.redirectTo("/pricing-v2");
}
