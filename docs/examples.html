Examples

Examples to learn how to use Abrantes in your websites.

Quick tips

All examples load the default AbrantesPlus.js (core + plugins). But you should generate a custom build with the plugins you need, instead of the default.

Some examples include Google Tag Manager or Google Analytics 4 snippets. Please replace the default IDs or remove the tracking scripts.
