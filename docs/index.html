Simple and powerful AB testing script

A lightweight, free and open-source JavaScript library to create advanced experiments on your website.

Abrantes integrates experiments with popular, often free, analytics software to give you a full picture of the user's behaviour.

Examples View on GitHub

Why abrantes?

Unlimited

No limits on experiments, variants, users, or pageviews. Run as many tests as you want.

Performance first

Lightweight (3.8Kb) and optimized for speed. Minimizes the "flash of switching content" without blank screen flicker.

Works with your tools

Seamlessly integrates with Google Tag Manager, Google Analytics, Microsoft Clarity, and more.

Completely free

No paid plans, no add-ons. Abrantes is 100% free software that you can use for any purpose.

Open source

Transparent and extensible. Licensed under LGPL v3. Modify it to fit your exact needs.

Privacy focused

Abrantes doesn't collect or store user data. It works with your existing analytics tools.

Integrates with your stack

Abrantes is designed to work with the tools you already use.

Google Tag Manager Google Analytics Mixpanel Hotjar Forms Matomo Microsoft Clarity Hubspot Enterprise
